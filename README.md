# SimDecisions Global Commons

Public repository of shared resources for SimDecisions users.

## Contents

### `/prompts`
Base system prompts for LLM interactions.
- `canvas-base.md` — Base prompt for canvas chat

### `/scenarios`
Ready-to-use process scenarios.
- `index.json` — Catalog of all available scenarios
- `/samples` — Example scenarios for learning
- `/templates` — Starting points for new processes

### `/schemas`
JSON schemas for validation.
- `ir-graph.schema.json` — IR Graph structure validation

## Usage

These resources are fetched by SimDecisions at runtime:
- Base prompt loaded when canvas chat initializes
- Scenario index shown in Global Commons browser
- Users can "Try" scenarios or "Copy to My Repo"
